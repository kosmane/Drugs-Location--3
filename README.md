# 🗄️ מאגר חומרים – PWA

אפליקציית חיפוש חומרים עם הצגת מיקום בארון.  
הנתונים נטענים אוטומטית מהריפו – אין צורך בהגדרה ידנית.

---

## 📁 מבנה הריפו

```
Drugs-Location--3/
├── index.html
├── manifest.json
├── sw.js
├── icons/
│   ├── icon-192.png
│   └── icon-512.png
├── images/
│   ├── 1.jpg       ← תמונת ארון מספר 1
│   ├── 2.jpg       ← תמונת ארון מספר 2
│   └── ...
└── 2024-01-15.xlsx ← קובץ האקסל (שם = תאריך עדכון)
```

---

## 📱 התקנה על הטלפון

**Android (Chrome):** תפריט ⋮ → "הוסף למסך הבית"  
**iPhone (Safari):** כפתור שיתוף ⬆️ → "הוסף למסך הבית"

---

## 📊 עמודות האקסל הנדרשות

| עמודה | שימוש |
|-------|-------|
| `תיאור חומר` | מוצג בכרטיס + חיפוש |
| `מיקום בארון` | **מספר בלבד** (1, 2, 3...) |
| `חומר פעיל` | מוצג בכרטיס + חיפוש |
| `שם מסחרי` | חיפוש בלבד |
| `Brand names` | חיפוש בלבד |
| `שם בעברית` | חיפוש בלבד |
| `תיאור קבוצת חומר` | מוצג בכרטיס |

**שם קובץ האקסל** = תאריך העדכון (מוצג בראש האפליקציה)

---

## 🖼️ תמונות מיקום

- שמור תמונות בתיקייה `/images/`
- **שם הקובץ = מספר המיקום** + `.jpg`
- לדוגמה: מיקום `5` → `/images/5.jpg`

---

## 🔄 עדכון נתונים

- **אקסל חדש**: העלה קובץ xlsx עם שם=תאריך חדש (מחק את הישן)
- **תמונה חדשה**: העלה/החלף קובץ jpg בתיקיית images
